How to Play?

STEP #1

Select a mode between Best Response Dynamics questions and Semantics flip cards.

STEP #2

If you selected Best Response Dynamics: Choose a level from Easy, Medium, and Hard.



If you selected Semantics flip cards: You can start learning by hovering/clicking the cards.

STEP #3

If you selected Best Response Dynamics: Answer the questions by identifying what the best response is to that question.



If you selected Semantics flip cards: Now you can read and be familiar with the Kapampangan words that you still don't know.
